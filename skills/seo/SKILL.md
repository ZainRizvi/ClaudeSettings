# SEO Content Optimization

## Overview

Comprehensive SEO toolkit covering the full content optimization lifecycle. This skill routes to specialized sub-skills based on your task.

## When to Use

**Content Creation:**
- Writing new SEO-optimized articles → `seo:write`
- Planning content strategy/outlines → `seo:plan`
- Creating meta titles/descriptions → `seo:meta`

**Content Optimization:**
- Analyzing keyword usage/density → `seo:keywords`
- Optimizing content structure/headers → `seo:structure`
- Formatting for featured snippets → `seo:snippets`

**Content Maintenance:**
- Auditing existing content quality → `seo:audit`
- Refreshing outdated content → `seo:refresh`
- Detecting keyword cannibalization → `seo:cannibalization`
- Building E-E-A-T/authority signals → `seo:authority`

Invoke the appropriate sub-skill for your task. Each sub-skill contains specialized prompts and frameworks.

**Example:** "I need to write a new blog post about podcast hosting" → Use `seo:write`

**Example:** "Check if these two pages compete for the same keywords" → Use `seo:cannibalization`
