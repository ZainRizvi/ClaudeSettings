# Meta Tag Optimization

## Optimization Rules

**URLs:**
- Under 60 characters
- Hyphens, lowercase only
- Primary keyword early
- Remove stop words when possible

**Title Tags (50-60 chars):**
- Primary keyword in first 30 characters
- Emotional triggers/power words
- Numbers/year for freshness
- Brand placement strategy

**Meta Descriptions (150-160 chars):**
- Include primary + secondary keywords
- Action verbs and benefits
- Compelling CTAs
- Special characters for visibility (optional)

## Output Format

```
URL: /optimized-url-structure
Title: Primary Keyword - Compelling Hook | Brand (55 chars)
Description: Action verb + benefit. Include keyword naturally. Clear CTA. (155 chars)
```

## Deliverables

- Character count validation
- A/B test variations (3 minimum)
- Power word suggestions
- Schema markup recommendations
